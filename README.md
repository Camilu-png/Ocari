# Ocari
Aprende a tocar la ocarina de 12 hoyos de forma visual y sencilla.  Ocari es una aplicación móvil diseñada para músicos y entusiastas que desean dominar la ocarina de 12 hoyos. A través de una interfaz intuitiva, la app muestra las posiciones exactas de los dedos en tiempo real para cada nota de tus canciones favoritas.
