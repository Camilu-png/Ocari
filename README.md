# Ocari
##Aprende a tocar la ocarina de 12 hoyos de forma visual y sencilla.  

Ocari es una aplicación móvil diseñada para músicos y entusiastas que desean dominar la ocarina de 12 hoyos. A través de una interfaz intuitiva, la app muestra las posiciones exactas de los dedos en tiempo real para cada nota de tus canciones favoritas.

### Características principales

- Guía Visual en Tiempo Real: Visualización interactiva de la posición de los dedos para ocarinas de 12 hoyos.

- Biblioteca de Canciones: Incluye temas icónicos.

- Control de Aprendizaje: Ajusta la velocidad de reproducción para practicar a tu propio ritmo.

- Modo Próximas Notas: Anticipa los cambios de posición con una línea de tiempo de notas futuras.
